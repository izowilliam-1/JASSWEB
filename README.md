# JASSWEB - Juba Seventh Day Adventist Secondary School Website

Welcome to the official website of **Juba Seventh Day Adventist Secondary School (JASS)**!

## 🎯 About

JASWEB is a comprehensive school activities website that showcases all school information, activities, news, fees, and contact details for JASS.

### School Information
- **Full Name**: Juba Seventh Day Adventist Secondary School
- **Short Name**: JASS
- **Motto**: "Wholistic Education for Brighter Future"
- **Type**: Complete Secondary School
- **Classes**: S1 to S4
- **Curriculum**: South Sudan National Curriculum
- **Location**: Inside SDA Church Compound, SDA Roundabout, Munuki, Juba, Central Equatoria, South Sudan

## 🚀 Getting Started

### Option 1: View Locally
1. Clone this repository
2. Open `index.html` in your web browser
3. Navigate through the different sections

### Option 2: Deploy Online
You can deploy this website using:
- **GitHub Pages** (Free)
- **Netlify** (Free)
- **Vercel** (Free)
- **Any web hosting service**

### Steps to Deploy on GitHub Pages:
1. Go to repository Settings
2. Scroll to "GitHub Pages" section
3. Select "main" branch as source
4. Your site will be live at: `https://izowilliam-1.github.io/JASSWEB/`

## 📝 Customization

### Update Contact Information
Edit the **Contact Section** in `index.html`:
- Add phone number
- Update email addresses
- Change office hours

### Add More Activities
Add new activity cards in the **Activities Section**:
```html
<div class="activity-card">
    <div class="activity-icon">🎯</div>
    <h3>Activity Name</h3>
    <p>Activity description here.</p>
</div>
```

### Update News
Add new news items in the **News Section**:
```html
<div class="news-card">
    <div class="news-date">Date</div>
    <h3>News Title</h3>
    <p>News description here.</p>
</div>
```
